## Overview

Solar-X is an interactive educational platform specifically designed for children, aiming to transform the study of the solar system into a captivating, accessible, and fun learning experience. The project effectively demonstrates proficiency in clean frontend development by using dynamic elements to engage young learners with complex astronomical concepts.

## Key Features

**Interactive Planetarium** : A core feature that allows children to visually explore planetary orbits, relative sizes, and spatial relationships within the solar system.
**Engaging 'Fun Facts' and Quizzes** : Presents fascinating, bite-sized facts and integrates interactive quizzes to test and reinforce knowledge retention in an enjoyable way.
**"What If" Scenarios** : Features thought-provoking content designed to stimulate curiosity and encourage critical thinking about hypothetical cosmic events.
**"Responsive Design** : Fully responsive interface built with Bootstrap, ensuring optimal viewing and functionality across various devices (mobile, tablet, and desktop).
